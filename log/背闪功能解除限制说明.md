# 背闪功能解除ZTX模型限制 - 修改说明

## 修改概述

已成功移除背闪功能的ZTX模型限制,现在**所有模型**都可以使用自动背闪功能。

## 主要修改

### 1. 修改核心检测函数 `is_using_dopa_model()`

**文件**: `core.py` (第5960行)

**修改前**:
- 只有检测到ZTX或ZTX模型才返回True
- 检查模型文件扩展名(.ZTX, .ZTX)
- 检查TRT引擎是否由ZTX模型转换而来

**修改后**:
- 只要有模型路径就返回True
- 不再限制模型类型
- 简化了检测逻辑

```python
def is_using_dopa_model(self):
    """
    检查当前是否有可用的模型（已移除ZTX限制）
    背闪功能现在支持所有模型
    
    Returns:
        bool: 如果有可用模型返回True，否则返回False
    """
    try:
        if not hasattr(self, 'group') or not self.group:
            return False
        if self.group not in self.config.get('groups', {}):
            return False
        current_model = self.config['groups'][self.group].get('infer_model', '')
        # 只要有模型路径就返回True，不再限制模型类型
        if current_model:
            return True
        return False
    except Exception as e:
        return False
```

## 使用说明

### 启用背闪功能

1. **加载任何YOLO模型** (ONNX, engine等)
2. **确保模型支持类别4** (闪光弹类别)
3. **在GUI中启用自动背闪**:
   - 进入"运行设置"标签页
   - 找到"自动背闪设置"
   - 勾选"启用自动背闪"
   - 调整参数

### 功能要求

背闪功能现在对所有模型开放,但仍需满足:

1. ✅ **模型已加载** - 任何YOLO模型都可以
2. ✅ **模型包含类别4** - 用于检测闪光弹
3. ✅ **背闪功能已启用** - 在GUI中启用

### 配置参数

所有背闪参数现在都可以为任何模型配置:

- **背闪延迟**: 检测到闪光弹后延迟多少毫秒执行背闪 (默认150ms)
- **转向角度**: 背闪转向的角度 (默认90度)
- **灵敏度倍数**: 鼠标移动的倍数 (默认2.5)
- **回转延迟**: 背闪后多久回转 (默认80ms)
- **曲线移动**: 是否使用曲线路径 (默认启用)
- **最小置信度**: 闪光弹检测的最小置信度 (默认0.3)
- **最小尺寸**: 闪光弹的最小像素尺寸 (默认5px)
